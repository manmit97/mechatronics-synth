# 3D Parts Interaction and Visualization Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Improve the 3D Viewer by rendering accurate part features using Constructive Solid Geometry (CSG), enabling part movement/dragging, and adding zoom UI controls.

**Architecture:** We will use `@react-three/csg` to subtract holes/slots and add bosses/ribs based on `ProceduralConfig.features`. We'll use `@react-three/drei`'s `TransformControls` to allow moving parts around, hooked up to `useConfiguratorStore`'s `updatePartTransform`. Finally, we will add HTML-based zoom controls that modify the R3F camera to make zooming explicit and accessible.

**Tech Stack:** React, Next.js, Three.js, React Three Fiber, React Three Drei, React Three CSG.

### Task 3: Enable Grabbing and Moving Parts

**Files:**
- Modify: `src/components/three/Scene3D.tsx`

**Step 1: Wrap selected parts in `TransformControls`**

In `Scene3D.tsx`, check if `isSelected`. If so, wrap the part in `<TransformControls mode="translate">`. Listen to the `onMouseUp` event or `onChange` on the control to read the `meshRef.current.position` and call `updatePartTransform(placed.instanceId, { position: [...] })`. Add `makeDefault` to `OrbitControls` so it correctly yields when dragging the TransformControls.
